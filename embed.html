Embedding in iframe

This allows you to include the website (e.g. http://chrisnorth.github.io/plotgw/index.html) in your own page without having to have any of the source files

Advantages

  • Doesn't require any direct inclusion of libraries, javascript, css etc.
  • Requires no updating.
  • Good if you only have access to HTML

Disadvantages

  • Will change/break whenever the webpage within the iframe breaks/changes
  • You have no control over what's in the iframe
  • Some system managers may ban iframes
  • Requires a web connection

Required code

In the page body:

    <iframe src="https://chrisnorth.github.io/plotgw" width="900px" height="600px></iframe>

Including directly in your own page

Advantages

  • You have full control (in principle) over what's in the included content
  • Won't break/change if something else breaks/changes
  • Requires no web connection if run locally (beware browser security issues)

Disadvantages

  • Requires the inclusion of javascript libraries, css etc. in the webpage
  • Will require updating if anything changes
  • Stylying that depends on screen size might look a bit weird odd

Required Code

Clone/download the Github repository

In the page header:

    <link rel="icon" type="image/vnd.microsoft.icon" href="img/favicon-32x32.png" />
    <link rel="stylesheet" href="css/style-d3.css" type="text/css" />
    <script type="text/javascript" src="js/d3.v3.min.js"></script>
    <script type="text/javascript" src="js/clipboard-1.7.1.min.js"></script>
    <script type="text/javascript" src="js/FileSaver.js"></script>
    <script type="text/javascript" src="js/gwcat.js"></script>
    <script type="text/javascript" src="js/plotgw-d3.js"></script>

In the page body:

    <div id="plotgw-cont" style="width:900px;height:600px;border:1px solid black"></div>
    <script type="text/javascript">
        var gwcat = new GWCatalogue({'holderid':'plotgw-cont','clearhtml':false})
    </script>
